# Keg.I18n

`widgets` `datafields` `apps` `watchfaces`

#### Sub-module(s): None

***

### Description

Get a string from resources. If a string resource ID is not found a blank string is returned.

***

### Module Details

#### t() as _string_

```js
using Keg.I18n;

var string = I18n.t(:SomeStringId);
```
