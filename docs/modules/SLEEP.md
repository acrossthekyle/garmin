# Keg.Sleep

`watchfaces`

#### Sub-module(s): None

***

### Description

Determine if the user is awake or not based on device settings for sleep and wake time(s).

***

### Module Details

#### isUserAwake() as _boolean_

```js
using Keg.Device;

if (Sleep.isUserAwake()) {
  // do something
} else {
  // do something else
}
```
